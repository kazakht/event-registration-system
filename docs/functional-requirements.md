# 機能要件定義

## 機能要件一覧

本ドキュメントは、業務要件を実現するために必要なシステム機能を定義します。

| ID     | 大機能           | 小機能                   | 内容（詳細）                                                                                       | 対応する業務要件ID |
| ------ | ---------------- | ------------------------ | -------------------------------------------------------------------------------------------------- | ------------------ |
| FR-001 | イベント検索     | 検索フォーム             | ユーザーがキーワードを入力できる検索フォームをフロントエンドに実装する                             | BR-001             |
| FR-002 | イベント検索     | キーワード検索API        | イベント名・概要に対してキーワード検索を行い、該当するイベント一覧を返却するAPIを提供する         | BR-001             |
| FR-003 | イベント一覧表示 | イベント一覧取得API      | 開催予定のイベント一覧（タイトル、日時、場所の概要）を取得するAPIを提供する                       | BR-002             |
| FR-004 | イベント一覧表示 | イベント一覧画面         | 取得したイベント一覧を一覧形式で表示する画面を実装する                                             | BR-002             |
| FR-005 | イベント詳細表示 | イベント詳細取得API      | 指定されたイベントIDに基づき、イベント詳細情報（日時、場所、概要、チケット情報）を返却するAPIを提供 | BR-003～BR-007     |
| FR-006 | イベント詳細表示 | イベント詳細画面         | 取得したイベント詳細を表示する画面を実装する                                                       | BR-003             |
| FR-007 | イベント詳細表示 | 開催日時表示             | イベント詳細画面に開催日時を表示する                                                               | BR-004             |
| FR-008 | イベント詳細表示 | 場所表示                 | イベント詳細画面に開催場所を表示する                                                               | BR-005             |
| FR-009 | イベント詳細表示 | 概要表示                 | イベント詳細画面にイベント概要を表示する                                                           | BR-006             |
| FR-010 | イベント詳細表示 | チケット情報表示         | イベント詳細画面にチケット種別、価格、残数を表示する                                               | BR-007             |
| FR-011 | 参加登録         | 参加登録API              | メールアドレスとチケット種別を受け取り、参加登録を行うAPIを提供する                                | BR-008～BR-010     |
| FR-012 | 参加登録         | 在庫確認・減算処理       | チケットの残数を確認し、在庫がある場合のみ登録を許可し、在庫を減算する                             | BR-007, BR-008     |
| FR-013 | 参加登録         | 参加登録フォーム         | メールアドレス入力欄とチケット選択UI（ドロップダウン or ラジオボタン）を持つフォームを実装する    | BR-009, BR-010     |
| FR-014 | 参加登録         | バリデーション           | メールアドレス形式チェック、チケット選択必須チェックを実装する                                     | BR-009, BR-010     |
| FR-015 | 参加登録         | 登録完了画面             | 参加登録成功後に表示する完了画面を実装する                                                         | BR-011             |
| FR-016 | 参加登録         | 登録完了メール送信       | 参加登録完了後、入力されたメールアドレスに確認メールを送信する機能を実装する                       | BR-012             |

## 機能の優先度

### MVP (Minimum Viable Product) 必須機能

- FR-001, FR-002: キーワード検索
- FR-003, FR-004: イベント一覧表示
- FR-005～FR-010: イベント詳細表示
- FR-011～FR-016: 参加登録

### Phase 2 以降
- FR-020, FR-021: エラーハンドリングの詳細化
